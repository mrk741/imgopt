How to Use the Image Optimizer

  • Upload images — drag and drop or click the upload area.
  • Adjust settings — set max width/height, choose output format, and use the quality slider (percentage shown live).
  • Background Removal — by default, the tool removes people backgrounds on-device using MediaPipe. You can disable it from the dropdown if not needed.
  • Process Images — click “Process Images” to begin optimization. The progress bar shows overall status.
  • Download ZIP — once done, click to download all optimized images in one archive.
  • Clear Queue — removes all files so you can start fresh.

Supported Features

  • Image resizing (maintains aspect ratio)
  • Format conversion: JPEG, PNG, WebP, AVIF
  • Adjustable compression quality
  • On-device background removal for people
  • Bulk ZIP export
